# Plan 03-01: Jira Adapter Implementation

<objective>
Implement the `JiraAdapter` using `httpx` and `tenacity` to fulfill the `JiraPort` protocol, including proper rate-limit backoff and transparent pagination.
</objective>

<interfaces>
- `JiraAdapter` must inherit from `JiraPort`.
- Constructor: `__init__(self, base_url: str, email: str, api_token: str)`
- `fetch_ticket(self, ticket_key: str) -> JiraTicket`
- `fetch_ticket_comments(self, ticket_key: str) -> list[str]`
- `fetch_linked_issues(self, ticket_key: str) -> list[str]`
</interfaces>
